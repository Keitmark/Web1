Назначение проекта

Проект представляет собой многостраничный информационный сайт, посвящённый городу Мариуполь. Его цель — предоставить пользователю основные сведения об истории города, его культуре, инфраструктуре и современных событиях. Сайт демонстрирует навыки веб-разработки: верстку, работу с изображениями, создание формы обратной связи и структурирование большого объёма информации.

Использованные технологии

HTML5 — для создания структуры страниц и семантической разметки.

CSS3 — для визуального оформления, адаптивности и стилизации элементов.

Flexbox / Grid (если использовал) — для построения адаптивного макета.

Google Maps iframe — для отображения карты города.

Встроенная HTML5-валидация — для проверки формы обратной связи.

Структура страниц

Главная страница — краткое описание города, фотогалерея, блок актуальных событий.

О разработчике — фотография, биография, навыки с диаграммами.

История города — таблица с ключевыми историческими событиями.

Инфраструктура — географическое положение, административное деление, карта, транспорт.

Культура и искусство — театры, музеи, галереи, культурные деятели.

Форма обратной связи — поля имени, email, сообщения, кнопка отправки.

Как запустить сайт локально

Скачайте папку проекта на компьютер.

Убедитесь, что структура выглядит примерно так:

/project-folder
    index.html
    about.html
    history.html
    culture.html
    infrastructure.html
    contact.html
    /css
        style.css
    /img
        *.jpg / *.png


Откройте файл index.html двойным щелчком — он автоматически откроется в браузере по умолчанию.

Перемещайтесь по сайту с помощью навигационного меню.
